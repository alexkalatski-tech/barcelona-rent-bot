# 🏠 Barcelona Rent Bot

Telegram bot that monitors rental apartments in Barcelona and nearby cities.

## Features

- Monitor new listings
- Detect price drops
- Detect removed listings
- Telegram notifications
- Apartment scoring
- SQLite database
- Multi-source architecture
